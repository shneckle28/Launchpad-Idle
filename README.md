# Launchpad Pro Idle Controller

Turn your **Novation Launchpad Pro** into a live animated LED display.

Launchpad Pro Idle Controller is a **Windows-only desktop app** that runs smooth full-grid idle animations directly on your Launchpad Pro. It is distributed as a **prebuilt `.exe`**, so you can download it, launch it, and use it without installing Python or setting up the source code.

If you want your Launchpad to stay visually active on your desk when you are not using it, this app is built for exactly that.

---

## Download

Download the latest **Windows `.exe`** from the **Releases** section of this repository.

---

## Quick Start

1. Download the latest `.exe`
2. Plug in your **Novation Launchpad Pro**
3. Run the app
4. Sellect your Midi
5. Enjoy the animations

## What It Does

This app sends animated RGB lighting effects across the full **10x10 Launchpad Pro grid**, turning the controller into a generative LED display.

It includes multiple built-in animations and color presets, ranging from soft ambient effects to more energetic and hypnotic patterns.

The goal is simple: make your Launchpad Pro look alive even when it is idle.

---

## Features

- **Windows-only prebuilt `.exe`**

## Troubleshooting

### The app opens but the Launchpad does not light up
Make sure the Launchpad Pro is connected and not already being used by another MIDI application.

### Nothing happens when I start the app
Close any DAWs, MIDI routing tools, or background software that may already be using the Launchpad.

### Windows shows a warning before opening the app
Because this is an independently distributed executable, Windows SmartScreen or antivirus software may sometimes show a warning. Only continue if you downloaded the file from the official repository or release page.

## Why This Exists

Most MIDI controllers sit dark when they are not being played.

This app gives the Launchpad Pro a second life as an ambient LED display, desk centerpiece, and programmable visual toy. It is meant for people who like MIDI gear, generative visuals, and creative hardware projects.

## For Developers

Although the app is distributed as a bundled Windows executable, the underlying project is built around modular animation classes and palette presets, making it easy to extend with new visual modes later.

Each animation generates a 10x10 RGB frame, which keeps the system straightforward to customize and expand.
